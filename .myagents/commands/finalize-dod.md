# Finalize DoD

Update `dod.md` to match the final result, then store it.

## Input

- Final result
- `dod.md`

## Output

- `.myagents/artifacts/dod/<task-summary>-<timestamp>.md`

## Steps

- [ ] Update `dod.md` from the final result
- [ ] Ensure `Open Questions` is empty
- [ ] Save a copy to `.myagents/artifacts/dod/<task-summary>-<timestamp>.md`

## Rules

- Keep it minimal
- Use a short filesystem-safe task summary
- Use a sortable timestamp such as `YYYYMMDD-HHMMSS`
